# Politique de Confidentialité - NuovaLingua

**Dernière mise à jour :** 6 novembre 2025

## Introduction

NuovaLingua ("nous", "notre", "l'application") s'engage à protéger votre vie privée. Cette politique de confidentialité explique comment nous collectons, utilisons et protégeons vos informations lorsque vous utilisez notre application mobile.

## Informations que nous collectons

### Données stockées localement

NuovaLingua fonctionne principalement en mode hors ligne. Les données suivantes sont stockées **uniquement sur votre appareil** :

- **Dictionnaire personnel** : Mots que vous ajoutez, leurs traductions et votre progression
- **Textes sauvegardés** : Articles et textes que vous importez depuis le web
- **Conversations** : Historique de vos conversations avec l'IA
- **Statistiques** : Votre progression, temps d'étude, nombre de mots appris
- **Préférences** : Paramètres de l'application, notifications, thèmes sélectionnés
- **Clés API** : Si vous configurez vos propres clés API (OpenAI, Google TTS), elles sont stockées localement sur votre appareil

### Synchronisation Firebase (optionnelle)

Si vous activez la synchronisation Firebase dans les préférences :

- Vos données sont synchronisées avec Firebase Firestore
- L'authentification est anonyme (aucune information personnelle requise)
- Vous pouvez désactiver la synchronisation à tout moment
- Vos données restent sous votre contrôle

### Données transmises aux services tiers

Lorsque vous utilisez certaines fonctionnalités, des données peuvent être envoyées à :

- **OpenAI** : Si vous configurez une clé API, vos messages de conversation sont envoyés à OpenAI pour générer des réponses. Consultez la [politique de confidentialité d'OpenAI](https://openai.com/policies/privacy-policy)
- **Google Text-to-Speech** : Si vous configurez une clé API, les mots à prononcer sont envoyés à Google. Consultez la [politique de confidentialité de Google](https://policies.google.com/privacy)
- **Firebase** : Si vous activez la synchronisation, vos données sont stockées sur les serveurs Firebase. Consultez la [politique de confidentialité de Firebase](https://firebase.google.com/support/privacy)

**Important** : Nous n'avons pas accès à vos clés API. Elles sont stockées localement sur votre appareil et vous êtes responsable de leur sécurité.

## Comment nous utilisons vos informations

- Pour fournir les fonctionnalités de l'application (apprentissage, révisions, conversations)
- Pour améliorer votre expérience d'apprentissage
- Pour synchroniser vos données entre appareils (si Firebase est activé)
- Pour générer du contenu pédagogique adapté à votre niveau

## Partage de données

Nous ne vendons, ne louons ni ne partageons vos données personnelles avec des tiers, sauf :

- Si vous activez Firebase, vos données sont synchronisées avec Firebase (authentification anonyme)
- Si vous utilisez des services tiers (OpenAI, Google TTS), les données nécessaires sont transmises conformément à leurs politiques de confidentialité

## Sécurité

- Toutes les données sont stockées localement sur votre appareil par défaut
- Les clés API sont stockées de manière sécurisée dans le stockage local de l'appareil
- La synchronisation Firebase utilise une authentification anonyme sécurisée
- Nous ne collectons pas d'informations d'identification personnelle sans votre consentement explicite

## Vos droits

Vous avez le droit de :

- Accéder à toutes vos données stockées localement dans l'application
- Supprimer toutes vos données via les paramètres de l'application
- Désactiver la synchronisation Firebase à tout moment
- Ne pas utiliser les services tiers (OpenAI, Google TTS) si vous préférez

## Cookies et technologies similaires

NuovaLingua n'utilise pas de cookies. L'application fonctionne principalement hors ligne.
